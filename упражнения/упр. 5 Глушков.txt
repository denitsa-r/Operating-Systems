grep - ако не подадем име на файл почва да дири в текущата директория рекурсивно

оператори 

<(...) -> повечето процеси си имат собствени темп файлове, които пазят output-а им, като използваме няква програма се създава нов темп файл в /tmp директорията, този оператор прави нещо като събшела, ама по-скоро вместо аутпута подава директорията на тмп файла след изпълнението на командата 
comm -1 -2 <(sort f1) <(sort f2)

! когато ни трябва файл от няква команда, вместо събшел ще използваме тази команда <(...)

use-case:
$(...) -> когато ни трябва да вземем съдържанието на дадена команда
<(...) -> когато ни трябва файл със съдържание

ще си говорим за процеси:
$ps -> показва моите процеси 
$ps -e -> показва всички процеси 

$ps -> изкарва таблица:
PID -> process id
TTY -> technical typewriter, интерактва с инпут/аутпута
TIME -> колко време рънва процеса 
CMD -> име на командата 
* готиното тука е аутпут формати 
най-често ще ползваме 
$ps -o - задаваме собствен формат за изход
